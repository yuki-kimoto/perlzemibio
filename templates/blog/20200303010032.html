塩基配列の長さを取得

塩基配列の長さを取得するにはlength文を使用します。 
length 文字列互換型;
塩基配列から、文字列の長さを取り出すサンプルです。 
my $sequence = "ATGCAGCCC";
my $length = length $sequence;

塩基配列の長さを取得するPerlから呼び出すサンプル

# Sequence.spvm
package Sequence {
  has sequence : string;

  sub new : Sequence ($sequence : string) {
    my $obj = new Sequence;
    $obj->{sequence} = $sequence;
    return $obj;
  }

  sub length : int ($self : self) {
    return length $self->{sequence};
  }
}

use strict;
use warnings;
use SPVM "Sequence";

my $sequence = Sequence->new("ATGCAGCCC");
print $sequence->length, "\n";
# Output
# 9
